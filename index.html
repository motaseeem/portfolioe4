Description :

Contexte: Après la fusion de deux laboratoires pharmaceutiques pour créer le laboratoire Galaxy-Swiss Bourdin, nous avons dû développer une application web en PHP de gestion de fiches de frais des visiteurs médicaux qui devront être traitées et validées, ou non, par des comptables tous les mois. MonPetitMVC

Description :

Le projet "Mon petit MVC" en PHP a été une opportunité pour explorer la structure MVC (Modèle-Vue-Contrôleur) dans le développement web. En se basant sur le modèle, la vue et le contrôleur, nous avons abordé divers sujets dans un contexte très similaire à la vie quotidienne. Cela nous a permis de comprendre le fonctionnement de cette architecture et de l'appliquer à des scénarios concrets, enrichissant ainsi notre expérience en développement web avec PHP.
